Ventana Lugar

En esta ventana se podrá crear nuevos Lugares para la aplicacion. Tambien aparece una lista de los Lugares existente, los cuales se podrán seleccionar y añadir al grupo.


Al pulsar el botón "Actualizar Lugares", se eliminarán todos los lugares asignados al grupo en la base de datos y se crean nuevas relaciones entre el grupo del usuario y el lugar o lugares seleccionados. Para seleccionar más de un lugar a la vez usar la tecla ´control´.

Se puede acceder a esta ventana desde el submenu "Menu" -> "Lugares".


Esta ventana solo es accesible para usuarios con perfil de administrador.
